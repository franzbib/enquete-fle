# Ajustement narratif Delphine V0.4

## Objectif

La V0.4 remplace le personnage generique du secretariat par Delphine dans le scenario `Le dossier disparu`.

Ce remplacement mobilise un personnage recurrent de la reserve parce qu'il est pertinent pour l'enquete : Delphine travaille au secretariat et son role administratif soutient naturellement la disparition d'un dossier.

## Regle de reserve

Cette utilisation ne rend pas les autres personnages recurrents obligatoires.

Un personnage recurrent doit etre integre seulement s'il sert :

- le lieu ;
- la situation narrative ;
- la progression ludique ;
- l'objectif FLE/FOU.

## Impact scenario

ID final utilise : `delphine`.

Textes ajustes :

- fiche personnage de Delphine ;
- planning du secretariat ;
- temoignage de Chen ;
- temoignage de Fahad ;
- brouillon de mail ;
- element de preuve de l'interruption a 16h35 ;
- option non correcte de l'enigme 2.

## Points preserves

La contradiction autour de Fahad reste inchangee :

- Fahad affirme etre parti tout de suite apres son passage au secretariat ;
- l'historique d'impression indique une session a son nom a 16h48 ;
- l'interpretation correcte reste prudente.

L'inventaire reste inchange :

- le badge visiteur reste au secretariat ;
- il n'est pas presente comme un badge volontairement laisse par Delphine ;
- il conserve seulement sa fonction d'acces a la salle informatique.
